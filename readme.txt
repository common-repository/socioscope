Enables Facebook, Twitter and Email Share Button on Posts


== Features ==

* No Extra Settings/Configuration.
* Lightweight.
* Easy to use.

** Social Share Button **

* Facebook
* Twitter
* Email


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
